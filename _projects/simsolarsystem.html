--- title: Sim Solar System image: SimSolarSystem.png ext_url: https://github.com/apclemens/SimSolarSystem ext_url_name: Github link layout: project content_image: sss.gif --- This was an undergrad project where I simulated the solar system using the Runge-Kutta method. I downloaded the orbital constants from the Jet Propulsion Laboratory for the planets, several moons, and 20 objects in the asteroid belt. The simulation was run with different timestep values to see the effect of less precision on the stability of the orbits. The conclusion is moons don't seem to like the larger timesteps.
